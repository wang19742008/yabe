## Redis存储结构

* 通讯录fans关系
```
通讯录中电话 -> 用户电话
如：
t_1360003 -> [135000,136000,136001]
```

* 用户fans关系
```
用户ID->粉丝ID
如:
f_100 -> [101]
```
* 用户follow关系
```
用户ID->关注人ID
如:
g_101 -> [100]
```
* 电话到Uers的映射
```
用户电话->用户ID
如:
a_1360003 -> 100
```
* 商品发布列表
```
用户ID->商品ID集合
如:
p_100 -> [111,222] 
```
* 商品推送列表
```
用户ID->推送商品ID集合
如:
c_100 -> [111,222]
```
